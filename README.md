# 📝 To-Do App

A modern and responsive **To-Do Application** built using **HTML, CSS, and Vanilla JavaScript**. The app helps users manage daily tasks with features like editing, searching, filtering, dark mode, and persistent storage using Local Storage.

---

## 🚀 Features

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete individual tasks
- 🧹 Remove all tasks
- ✔️ Mark tasks as completed
- 🔍 Live search
- 📂 Filter tasks (All, Pending, Completed)
- 📊 Task statistics (Total, Completed, Remaining)
- 🌙 Dark / Light mode
- 💾 Save tasks using Local Storage
- ⌨️ Keyboard shortcuts (Enter to save, Escape to cancel)

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
